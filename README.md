## Run Locally

Clone the project

```bash
  git clone https://github.com/KobyEthan/note-taker-app.git note-taker
```

Go to the project directory

```bash
  cd note-taker
```

Install dependencies

```bash
  npm install
```
Run project
```bash
  npm start
```
